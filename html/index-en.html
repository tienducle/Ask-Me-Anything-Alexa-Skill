Overview

Using the form below, you can configure your own API key (OpenAI or Anthropic) to be used with the skill. Say "Register" to obtain your personalized, generated login credentials and enter them in the form below.

For more information about API keys, please consult the OpenAI documentation or Anthropic documentation.

Encryption Details

Your API key will be encrypted with your personal AWS UserId and an additional Salt.

When you open the skill, the Lambda backend decrypts your API key into memory to use it for the API requests.
