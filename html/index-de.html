Übersicht

Über das folgende Formular, kannst du deinen eigenen API-Schlüssel (OpenAI oder Anthropic) mit dem Skill verknüpfen. Sage "Registrieren" um deine personalisierten Anmeldedaten anzuzeigen und gib sie unten ein.

Für weitere Informationen zu den API-Schlüsseln, schaue bitte in der OpenAI Dokumentation bzw. Anthropic Dokumentation nach.

Informationen zur Verschlüsselung

Dein API-Schlüssel wird mit deiner persönlichen AWS UserId und einem zusätzlichen Salt verschlüsselt.

Wenn du den Alexa Skill aufrufst, wird das Lambda Backend den API-Schlüssel im Speicher entschlüsseln, um ihn für die API requests zu verwenden.

Einstellungen updaten

Um deinen API-Schlüssel zu löschen, sende das Formular einfach mit leerem "API Key" Feld ab.
